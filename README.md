Run following commands 

1. Pull the images from base folder pdf-rag
    > Docker compose up -d

2. Start server to run following command from server folder
    > pnpm run dev and
    > pnpm run dev:worker

3. Start Client to run following command from Client folder
   > pnpm run dev
